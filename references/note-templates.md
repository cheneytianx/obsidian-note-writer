# 笔记模板库

根据内容类型选择对应模板，可参考。可以根据用户偏好来调整。

---

## 模板1：概念解释笔记

适用：解释一个技术概念、术语、算法

```markdown
---
tags: [concept, TAG]
created: YYYY-MM-DD
source: conversation
---

# 概念名

> 一句话定义：...

## 核心思想

...

## 为什么重要

...

## 工作原理

...

## 与相关概念的关系

- 与 [[相关概念A]] 的区别：...
- 基于 [[基础概念B]] 构建：...

## 实际应用

...

## 参考

- [[来源笔记]]
```

---


## 模板选择决策

| 内容类型 | 模板 |
|---------|------|
| 解释一个概念 | 模板1：概念解释 |
| 读了一篇论文/文章 | 模板2：论文笔记 |
| 对话/讨论总结 | 模板3：对话总结 |
| 工具使用方法 | 模板4：工具笔记 |
| 今日记录 | 模板5：日记 |
| 不确定 | 模板3：对话总结（最通用） |
